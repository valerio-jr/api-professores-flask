# 🍎 API de Gerenciamento de Professores

Esta é uma API REST simples desenvolvida em **Python** utilizando o framework **Flask**. O objetivo do projeto é gerenciar um cadastro de professores, permitindo operações de criação, leitura, atualização e exclusão (CRUD), com armazenamento dos dados em um arquivo JSON local.

## 🚀 Tecnologias Utilizadas

* **Python 3**
* **Flask** (Framework Web)
* **JSON** (Persistência de dados)
